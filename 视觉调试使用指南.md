# RoboMaster视觉系统增强调试指南

## 🎯 增强可视化调试功能

### 功能特性

新的调试程序 `auto_aim_debug_mpc` 提供了丰富的可视化功能：

- ✅ **实时图像显示**：摄像头画面实时显示
- ✅ **多层信息叠加**：检测、跟踪、预测信息同时显示
- ✅ **键盘控制**：实时开关各种显示功能
- ✅ **调试信息面板**：云台状态、目标信息等
- ✅ **截图保存**：按S键保存当前帧

---

## 🚀 快速开始

### 1. 编译运行
```bash
cd build
make auto_aim_debug_mpc -j4
./auto_aim_debug_mpc configs/sentry.yaml
```

### 2. 可视化界面
程序会打开一个窗口显示：
- **窗口标题**：`RoboMaster Vision Debug`
- **实时画面**：摄像头图像 + 调试叠加信息

---

## 🎨 显示内容详解

### 1. YOLO检测结果（红色）
- **检测框**：红色矩形框住检测到的装甲板
- **置信度**：框上方显示检测置信度（如"0.95"）
- **类型标签**：框下方显示"SMALL"或"BIG"

### 2. 跟踪结果（绿色/橙色）
- **装甲板轮廓**：
  - **绿色**：当前被跟踪的装甲板（target.last_id对应）
  - **橙色**：其他候选装甲板
- **ID编号**：轮廓中心显示装甲板ID (0-4)

### 3. 预测瞄准点（红色）
- **瞄准轮廓**：粗红色线条
- **十字标记**：瞄准点中心有红色十字
- **位置**：基于预测轨迹计算的瞄准位置

### 4. 调试信息面板（左上角）
```
Gimbal: Y-2.3 P0.1 R1.2     # 云台姿态（度）
Targets: 1                     # 检测到的目标数量
Armors: 3                      # 检测到的装甲板数量
Bullet: 22.5 m/s               # 子弹速度
Mode: auto_aim                 # 当前工作模式
```

## ⌨️ 键盘控制

| 按键 | 功能 | 说明 |
|------|------|------|
| `D` | 切换检测显示 | 开关YOLO检测框显示 |
| `T` | 切换跟踪显示 | 开关装甲板轮廓显示 |
| `P` | 切换预测显示 | 开关瞄准点显示 |
| `I` | 切换信息面板 | 开关左上角信息显示 |
| `+`/`=` | 放大图像 | 图像缩放比例 +0.1 |
| `-`/`_` | 缩小图像 | 图像缩放比例 -0.1 |
| `S` | 保存截图 | 保存当前帧为JPG文件 |
| `Q` | 退出程序 | 安全退出调试程序 |

### 使用技巧
- **快速定位问题**：先按 `D` 关掉检测框，只看跟踪结果
- **性能测试**：观察帧率和延迟
- **调参验证**：实时观察参数调整效果

## 🔍 调试场景

```bash
# 运行调试程序，观察：
```

```bash
# 按 P 键观察瞄准点，验证：

# - 瞄准点是否领先目标位置
# - 预测轨迹是否合理
```

## 📊 数据流可视化

### PlotJuggler集成
程序同时发送数据到PlotJuggler（端口9870）：

#### 关键数据字段
- **位置数据**：`armor_x`, `armor_y`, `armor_z`
- **预测数据**：`x`, `y`, `z` (预测位置)
- **控制指令**：`cmd_yaw`, `cmd_pitch`
- **状态信息**：`w` (角速度), `nis` (卡方检验)

### 查看方法
```bash
# 启动PlotJuggler
plotjuggler &

# 在程序中：
# 1. 连接 UDP localhost:9870
# 2. 选择感兴趣的数据流
# 3. 观察曲线变化
```

## 🛠️ 高级功能

### 图像缩放
- **默认比例**：0.7倍（适合1920x1080显示）
- **调节范围**：0.3-2.0倍
- **用途**：在大屏幕上获得更好视野

### 截图保存
- **保存路径**：程序运行目录
- **文件名格式**：`debug_frame_{timestamp}.jpg`
- **用途**：记录关键帧用于分析

### 多目标显示
- **颜色编码**：
  - 绿色：当前跟踪目标
  - 橙色：候选目标
- **ID显示**：帮助理解目标切换逻辑

## 📈 性能监控

### 关键指标
- **帧率**：通过PlotJuggler观察时间戳间隔
- **延迟**：检测到目标到显示瞄准点的时间
- **稳定性**：跟踪轨迹的平滑度

### 优化建议
1. **GPU推理**：启用OpenVINO加速检测
2. **异步处理**：分离检测和显示线程
3. **图像压缩**：适当降低显示分辨率

## 📝 注意事项

1. **资源占用**：调试模式比正常模式消耗更多CPU
2. **网络延迟**：通过网络运行时可能有显示延迟
3. **存储空间**：频繁截图会占用磁盘空间
4. **安全退出**：使用Q键安全退出，避免数据丢失
